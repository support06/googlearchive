The iframe shim trick lets you overlay DIVs over the Street View panorama in all browsers (including Chrome and Firefox). The trick is to insert an iFrame above the Flash container but below the div one needs to protect. The iFrame should have the same size and position as the div. In the example above, the flash container has z-index 0, the iframe has z-index 1, and the div has z-index 2. This link is a good reference about iframe shims.
