Contents:
  maps - Google Maps API product

Both code bases are localized using a set of JSON formatted data files.  Each
locale is represented as a file named torchRelayJSON-<lang>_<country>.json. 
For example, the English (US) locale is stored within the
torchRelayJSON-en_us.json file.
